Program Kurulumu ve Kullanım Kılavuzu

1

Proje Dosyalarını Edinin

Tüm proje dosyalarını (ve web klasörünü) bilgisayarınıza indirin. Servis dosyasını çalıştırmak için gerekli kütüphaneleri kurun.

pip install psutil pystray google-api-python-client google-auth-oauthlib watchdog pillow
2

Programı Çalıştırın

Artık her iki işlevi de barındıran servis dosyasını çalıştırarak hem sistem tepsisini (tray) hem de bu arayüzle konuşan köprüyü aktif edebilirsiniz:

python pptx_archiver_service.py

Not: Program arka planda gizli çalışır. Sağ alttaki sistem tepsisinde (Tray) mavi yuvarlak ikonunu görebilirsiniz.

3

Google Web Client ID Ayarlayın

Google Cloud Console üzerinden oluşturduğunuz Web Uygulaması Client ID'nizi sağ üstteki çark ikonundan girin. Bu işlem tarayıcıdan Drive API'ye güvenli erişim için şarttır.

OAuth Arayüz Ayarları

Bu anahtar tamamen tarayıcınızda (localStorage) saklanır. Sunucularımıza veya GitHub'a asla gitmez.

Durum: Bağlantı test edilmedi

Ders notlarının (Anatomi, Biyokimya vb.) yapay zeka ile otomatik tasnif edilmesini sağlar. Yerel programın ayarlarına kaydedilir.

Google Cloud Kurulum Rehberi

  1. Google Cloud Console sitesine gidin.
  2. APIs & Services > Credentials sekmesine geçin.
  3. Create Credentials > OAuth client ID butonuna tıklayın.
  4. Application Type olarak Web Application seçin.
  5. Authorized JavaScript origins kısmına şunları ekleyin:
    • http://localhost:58291 (Lokal test için)
    • https://[kullanıcı-adınız].github.io (GitHub Pages için)
  6. Oluşturulan Client ID değerini kopyalayıp yukarıdaki kutuya yapıştırın ve Kaydet'e basın.
